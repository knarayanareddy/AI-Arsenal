## Overview

OpenAI's official developer forum, covering the OpenAI API, the ChatGPT Apps SDK, Codex, bug reports, and billing/API-credit issues.

## Who it's for

Developers building on the OpenAI API (Assistants/Responses API, Realtime API, image generation endpoints) and Codex users. Also useful for tracking OpenAI's own deprecation announcements, which are posted here directly (e.g. API deprecation notices for specific model versions).

## What you'll get

- Same-day responses on high-traffic issues (billing bugs, rate limits, API behavior changes) -- this is one of the most active vendor developer forums in the space
- Official deprecation and API-change announcements posted directly by OpenAI, sometimes ahead of broader press coverage
- A large archive of previously-answered questions covering nearly every common integration issue

## How to get value fast

Read the pinned "Read this before posting a new question" topic in the API category first -- it explicitly scopes what belongs in this forum (API/developer-platform discussion) versus what doesn't (general ChatGPT consumer questions, with narrow exceptions). Search for your specific error message, endpoint, or model name before posting; high-traffic recurring issues already have long threads. Tag your post with the relevant sub-topic (e.g. `api-billing`, `realtime`, `codex`) to reach the right readers.

## What to avoid

Don't post general ChatGPT consumer-app questions in the API-focused categories -- the forum's own pinned welcome post explicitly excludes this except in narrow cases, and off-topic posts tend to get low engagement.
