## Overview

The official Discourse forum for the Hugging Face ecosystem: transformers, datasets, PEFT, diffusers, the Hub, and deployment. Unlike chat servers, it is asynchronous, threaded, and fully web-searchable -- which makes it the durable Q&A record for the most widely used open-source ML stack.

## Who it's for

Anyone using Hugging Face libraries: beginners hitting first errors, practitioners debugging fine-tuning pipelines, and researchers with model- or dataset-hosting questions.

## What you'll get

- A deep, searchable archive of answered questions on transformers/PEFT/datasets errors -- often an exact match for your stack trace
- Library-specific categories where maintainers and experienced users answer
- Durable, linkable threads (unlike Discord history), useful for referencing solutions in your own docs or issues

## How to get value fast

Search the forum (and via web search, which indexes it) before posting -- exact error strings frequently hit answered threads. When posting, choose the specific library category and include a minimal reproducible snippet, library versions, and the full stack trace.

## What to avoid

Don't treat it as real-time support: answers arrive on forum timescales (hours to days). For interactive debugging, the Hugging Face Discord is faster; for confirmed library bugs, the relevant GitHub issue tracker is the right destination.
