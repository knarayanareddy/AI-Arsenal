# Entry Templates

Use these templates with the scaffold CLI or copy them manually into the correct `content/` folder.

```bash
pnpm run new:project
pnpm run new:tool
pnpm run new:paper
pnpm run new:tip
```

After creating an entry:

1. Fill every required frontmatter field.
2. Use only vocabulary from `TAXONOMY.md`.
3. Complete every Markdown section.
4. Run `pnpm run validate:all && pnpm run check:duplicates`.
