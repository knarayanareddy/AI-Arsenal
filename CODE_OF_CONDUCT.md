# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, caste, color, religion, or sexual identity and orientation.

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Our Standards

**Positive behavior includes:**

- Being respectful, direct, and constructive.
- Accepting feedback gracefully.
- Focusing on what is best for the community.
- Showing empathy toward other community members.
- Giving credit where it is due.
- Disagreeing politely when disagreeing is needed.

**Unacceptable behavior includes:**

- Harassment, trolling, or personal attacks.
- Discriminatory jokes or language.
- Publishing others' private information without permission.
- Sustained disruption of community spaces.
- Unwelcome sexual attention or imagery.
- Other conduct that would be inappropriate in a professional setting.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and PRs that violate this Code of Conduct. Serious or repeated violations may result in temporary or permanent bans from project spaces.

Enforcement is **proportionate to the conduct**, not the identity of the actor. We do not look the other way when a maintainer is the violator.

## Reporting

Report incidents privately to the lead maintainer via GitHub mention (`@knarayanareddy`) with the prefix `[CONDUCT]`.

**If the lead maintainer is the party you need to report**, escalate by:

1. Filing a private report on GitHub Security Advisories with title prefix `[CONDUCT ESCALATION]`.
2. If you cannot use GitHub, ask in any public issue for "an additional contact" — a co-maintainer will be designated in `docs/policies/continuity.md` if one exists.

We will respond within **7 days** and aim to resolve within **30 days**.

## Scope

This Code of Conduct applies to:

- All project spaces (issues, PRs, discussions, comments)
- Project representation (social media, conferences, mailing lists)
- Private communication when related to project work
