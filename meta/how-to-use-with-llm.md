# How to Use AI Arsenal with an LLM

## Recommended Context Order

1. `AGENT.md` — routing map.
2. `CONTEXT.md` — dense summary.
3. Relevant `content/**/_index.md` section files.
4. Specific entries or generated `data/*.json` files.

## Prompt Pattern

```text
Use AGENT.md as the navigation map and TAXONOMY.md as the vocabulary contract.
Answer using only AI Arsenal files unless I explicitly ask for outside research.
Route to the smallest relevant section first, then synthesize.
```

## Best Uses

- Stack selection.
- Architecture review.
- Tool comparison.
- Learning-path planning.
- Finding adjacent projects and alternatives.
