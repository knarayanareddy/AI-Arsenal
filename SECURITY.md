# Security Policy

AI Arsenal is a knowledge base. It may link to third-party tools and projects, but it does not audit or guarantee their security.

## Reporting Security Issues in This Repository

If you discover a security issue in repository automation, GitHub Actions configuration, scripts, or generated data, please report it privately to the maintainers rather than opening a public issue.

## Reporting Security Issues in Listed Tools

If you discover a vulnerability in a tool or project listed by AI Arsenal, report it to that upstream project using its responsible disclosure process. AI Arsenal can mark entries with warnings or status changes after upstream disclosure is handled.

## Scope

In scope:

- Vulnerabilities in repository scripts.
- Unsafe GitHub Actions behavior.
- Credential exposure risks.
- Malicious links or supply-chain concerns introduced into content.

Out of scope:

- Vulnerabilities in unrelated third-party projects, except for updating metadata or warnings in Arsenal entries.
